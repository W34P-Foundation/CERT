# AMY-CERT Image Assets

This folder contains image assets for the AMY-CERT website.

## Recommended Images

- `logo.png` - AMY-CERT logo (recommended size: 200x200px)
- `hero-bg.jpg` - Hero section background (recommended size: 1920x1080px)
- `favicon.ico` - Website favicon (recommended size: 32x32px)
- `og-image.jpg` - Open Graph social sharing image (recommended size: 1200x630px)

## Image Guidelines

- Use optimized images (compress before uploading)
- Prefer WebP format for better performance
- Include alt text for accessibility
- Maximum file size: 500KB for most images

## Adding Images

1. Optimize your images using tools like:
   - [TinyPNG](https://tinypng.com/)
   - [Squoosh](https://squoosh.app/)
   
2. Place images in this folder

3. Reference in HTML:
   ```html
   <img src="assets/images/your-image.png" alt="Description">
   ```
